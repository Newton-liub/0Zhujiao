# Excel 手动操作指南

目标：从原始成绩 Excel 中只保留 `学号/工号`、`学生姓名`、`班级`、`总分`，并整理成适合登分的新表。

## 一、处理前先备份

1. 找到原始 Excel 文件。
2. 复制一份。
3. 建议把副本命名为：`原文件名_处理用.xlsx`。
4. 后续只操作副本，不直接改原文件。

这样做的原因是：原始 Excel 里可能有 IP、提交时间、答题明细等敏感信息，处理过程中不要把这些内容带到最终表里。

## 二、单个 Excel 的处理方法

### 1. 找到关键列

打开 Excel 后，找到包含这些字段的表头行：

- `学号/工号`
- `学生姓名`
- `班级`
- `总分`

以当前样例表为例，真正有用的表头在第 2 行，学生数据从第 4 行开始。

### 2. 新建一个空白工作簿

1. 打开 Excel。
2. 新建一个空白工作簿。
3. 第一行填写表头：
   - A1：`学号/工号`
   - B1：`学生姓名`
   - C1：`班级`
   - D1：本次测试名称，例如 `阶段测试1`

注意：不要把原表的 `总分` 这个列名直接保留。它要改成本次测试名称。

### 3. 复制四列数据

从原始表复制：

- `学号/工号` 列 → 新表 A 列
- `学生姓名` 列 → 新表 B 列
- `班级` 列 → 新表 C 列
- `总分` 列 → 新表 D 列

复制时建议使用：

1. 复制原数据。
2. 到新表中右键。
3. 选择 `选择性粘贴`。
4. 选择 `值`。

不要直接复制整张表，也不要复制公式、批注、隐藏列。

### 4. 删除无关数据

最终新表只应该有这些列：

- `学号/工号`
- `学生姓名`
- `班级`
- 本次测试成绩列

不要保留：

- IP
- 状态
- 提交时间
- 获取时间
- 答题明细
- 题目分数
- 学生答题
- 备注、批注、隐藏列

## 三、多个 Excel 的处理方法

假设有三个文件：

- `单片机阶段测试1.xlsx`
- `单片机阶段测试2.xlsx`
- `单片机阶段测试3.xlsx`

最终新表应该是：

| 学号/工号 | 学生姓名 | 班级 | 阶段测试1 | 阶段测试2 | 阶段测试3 |
|---|---|---|---|---|---|
| 20243510 | 张三 | 电2404-1 | 80 | 85 | 90 |

### 推荐做法

1. 先按单个 Excel 的方式处理第一个文件。
2. 得到基础表：`学号/工号`、`学生姓名`、`班级`、`阶段测试1`。
3. 处理第二个文件，只复制其中的：
   - `学号/工号`
   - `总分`
4. 用 `学号/工号` 对齐，把第二个文件的总分填入新列 `阶段测试2`。
5. 后续文件重复这个流程。

如果不熟悉 `VLOOKUP` 或 `XLOOKUP`，建议直接使用本项目里的工具自动处理，避免手动错位。

## 四、手动合并时的公式方法

假设：

- 汇总表在 `汇总` 工作表。
- 第二次测试数据在 `阶段测试2` 工作表。
- `汇总` 表 A 列是学号。
- `阶段测试2` 表 A 列是学号，D 列是总分。

可以在汇总表的 `阶段测试2` 列输入：

```excel
=XLOOKUP(A2,阶段测试2!A:A,阶段测试2!D:D,"")
```

然后向下填充。

如果 Excel 版本没有 `XLOOKUP`，可以用：

```excel
=VLOOKUP(A2,阶段测试2!A:D,4,FALSE)
```

注意：使用公式合并完成后，建议复制结果列，再 `选择性粘贴` 为 `值`，避免后续源表移动导致公式失效。

## 五、重修学生怎么判断

班级格式示例：

```text
电2404-2
```

含义：

- `电`：专业或学院简称
- `24`：年级
- `04`：专业/方向编号
- `2`：班级号

判断方法：

1. 看所有学生班级中出现最多的年级。
2. 例如大多数是 `电24xx-x`，主年级就是 `24`。
3. 如果有人是 `电23xx-x` 或 `电22xx-x`，通常就是重修。
4. 重修学生放在最上方。
5. 重修学生内部仍然按班级和学号排序。

## 六、排序方法

### 1. 新增辅助列

为了手动排序稳定，建议在汇总表右侧新增几个辅助列：

- `是否重修`
- `班级排序`
- `学号排序`

### 2. 标记是否重修

如果主年级是 `24`，班级在 C 列，从第 2 行开始，可以在辅助列输入：

```excel
=IF(ISNUMBER(SEARCH("24",C2)),1,0)
```

含义：

- `0`：重修，排前面
- `1`：正常学生，排后面

如果班级格式稳定，例如都是 `电2404-1`，也可以更精确地取两位年级。

### 3. 排序

选中整张表，然后：

1. 点击 `数据`。
2. 点击 `排序`。
3. 第一排序条件：`是否重修`，升序。
4. 第二排序条件：`班级`，升序。
5. 第三排序条件：`学号/工号`，升序。
6. 确认排序。

排序完成后，可以删除辅助列。

## 七、最终检查清单

交付或上传前，检查最终 Excel：

- 是否只保留必要列。
- 是否没有 IP、时间、答题明细等敏感字段。
- 成绩列名称是否对应正确测试。
- 学号和姓名是否错位。
- 重修学生是否在最上面。
- 班级是否按 `电2404-1`、`电2404-2`、`电2405-1` 这种顺序排列。
- 同班级内学号是否从小到大。

## 八、防止数据泄露的关键原则

不要使用这些方式假装脱敏：

- 隐藏列
- 隐藏工作表
- 白色字体
- 单元格保护
- 把字体调小
- 用格式显示成星号

这些方式只是看不见，数据仍然在文件里。

真正安全的方式是：

1. 新建空白工作簿。
2. 只复制需要的数据。
3. 使用 `选择性粘贴` → `值`。
4. 删除所有无关字段。
5. 保存为新的 Excel 文件。
