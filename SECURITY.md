# Security Policy

## 数据边界

本项目面向成绩 Excel 汇总场景。公开仓库只应包含源码、文档和构建脚本，不应提交真实成绩表、学生名单、导出结果或任何包含个人信息的文件。

## 不应提交的内容

- 真实 Excel 成绩表、登分表、学生名单。
- 包含学号、姓名、班级、分数的样例数据。
- API token、密码、证书私钥、账号凭据。
- `dist/`、`build/` 等本地构建产物。

## 漏洞或数据泄露反馈

如果发现安全问题，请在 GitHub 仓库中提交 Security Advisory 或通过私有渠道联系维护者。请不要在公开 Issue 中粘贴真实学生数据、token 或完整敏感样本。
