# Excel 成绩汇总工具

这个工具用于把一个或多个考试 Excel 汇总成登分表。

## 输出内容

最终汇总表只保留：

- 学号/工号
- 学生姓名
- 班级
- 每个源 Excel 对应的一列成绩

例如导入：

- `单片机阶段测试1.xlsx`
- `单片机阶段测试2.xlsx`

输出列会类似：

- 学号/工号
- 学生姓名
- 班级
- 阶段测试1
- 阶段测试2
- 核查标记

## 核查标记

生成汇总表时会自动增加人工核查提示：

- 10 分制：标记 `10` 分和小于等于 `8` 分。
- 百分制：标记 `100` 分和小于等于 `60` 分。
- 其他分制：按检测到的满分和 60% 及格线做基础标记。
- 同一个学生某一次成绩明显低于自己的其他成绩时，会标记为核查。
- 如果该异常低分与前后同班同学成绩对调后更符合双方平时水平，会标记为重点核查。

导出结果中：

- `成绩汇总` 工作表最后一列会显示核查摘要。
- 被核查的成绩单元格会高亮，并带有批注说明。
- `核查明细` 工作表会列出级别、学生、成绩列、原因和建议。

## 辅助人工核查

生成汇总表后，可以点击 `辅助人工核查` 逐条处理待核查项目。

辅助核查窗口会按汇总表顺序显示：

- 学号/工号
- 班级
- 学生姓名
- 核查成绩列
- 原成绩
- 核查原因

左侧目录可以直接点击跳转。右侧底部支持按钮和快捷键：

- `1 核查无误`：记录结果并自动跳到下一条。
- `2 已修改（登记错误）`：输入修正后成绩，记录结果并自动跳到下一条。
- `3 未交卷/未给分`：记录结果并自动跳到下一条。
- `0 上一个`：返回上一条。

完成后会输出：

- `成绩汇总_核查报告.xlsx`：只抽出原本有核查标记的项目，并增加核查结果。
- `成绩汇总_已修正.xlsx`：如果存在登记错误，只替换对应学生、对应成绩列的修正成绩。
- `成绩汇总_核查报告_不含登记错误.xlsx`：可选生成，排除已修改的登记错误项。

## 排序规则

- 自动识别最多出现的年级作为主年级。
- 与主年级不同的班级视为重修，放在最上面。
- 其他学生按班级从小到大排列。
- 同班级内按学号/工号从小到大排列。

## 下载 Windows 版

不想运行源码的用户，可以在 GitHub Releases 下载 `ScoreTool-windows.zip`，解压后运行其中的 `ScoreTool.exe`。

## 直接运行源码

```powershell
python -m pip install -r requirements.txt
$env:PYTHONPATH = "$PWD\src"
python -m score_tool.app
```

## 打包 exe

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build_exe.ps1
```

打包后 exe 位置：

```text
dist\ScoreTool\ScoreTool.exe
```

## 使用步骤

1. 双击打开 `ScoreTool.exe`。
2. 点击 `添加 Excel`，选择一个或多个成绩表。
3. 检查每个文件右侧的成绩列名。
4. 如果列名不合适，选中文件后点击 `修改成绩列名`。
5. 点击 `选择保存位置`。
6. 点击 `预检`，确认表头、人数、警告信息。
7. 点击 `生成汇总表`。
8. 如有核查标记，点击 `辅助人工核查`，按快捷键逐条登记核查结果。
9. 打开输出文件检查结果。

## 数据安全

工具只读取并导出四类数据：

- 学号/工号
- 学生姓名
- 班级
- 总分

不会把 IP、提交时间、答题明细、题目分数等列写入汇总表。
